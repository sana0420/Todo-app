A sleek, responsive, and functional To-Do List application built with vanilla JavaScript. This app allows users to manage their daily tasks efficiently with category filtering and theme switching.

## ‚ú® Features

* **Task Management**: Add, edit, and delete tasks easily.
* **Categorization**: Organize tasks into categories like *Personal*, *School*, or *General*.
* **Search & Filter**: Quickly find tasks using the search bar or sort them by date (Newest/Oldest).
* **Theme Toggle**: Switch between **Dark Mode** and **Light Mode** for a comfortable viewing experience.

* **Responsive Design**: Optimized for both desktop and mobile viewing.

## üí° Usage

1. **Add a Task**: Type your task in the "Enter a task" field, select a category, and click **Add**.
2. **Search**: Use the search bar to filter through your existing tasks.
3. **Complete/Delete**: Use the green checkmark to complete or the red trash icon to remove a task.
4. **Theme**: Click the "Light Mode" button at the top left to toggle the interface colors.
