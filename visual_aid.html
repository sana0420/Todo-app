sequenceDiagram participant User participant HTML participant main.js participant TodoList participant UI participant localStorage User->>HTML: Enters task in input User->>HTML: Clicks Add button HTML->>main.js: add-btn click event main.js->>TodoList: addTodo(task) TodoList->>Todo: new Todo(text) TodoList->>TodoList: push to todos array TodoList->>localStorage: save() main.js->>UI: render() UI->>TodoList: getTodos() UI->>HTML: Update todo-list element

Sequence Diagram: Toggling Completion

sequenceDiagram participant User participant HTML participant UI participant TodoList participant localStorage User->>HTML: Clicks on task li HTML->>UI: li click event UI->>TodoList: toggleComplete(todo.id) TodoList->>TodoList: find and toggle completed TodoList->>localStorage: save() UI->>UI: render() UI->>HTML: Re-render list with updated class

Sequence Diagram: Searching Tasks

sequenceDiagram participant User participant HTML participant UI participant TodoList User->>HTML: Types in search-input HTML->>UI: input event UI->>UI: render() UI->>TodoList: getTodos() UI->>UI: filter todos by searchTerm UI->>HTML: Update todo-list with filtered items

This visual aid shows the key classes and workflows of the OOP To-Do app. The app uses localStorage for persistence, allowing tasks to survive page reloads.
